2-3. 객실 이동

객실 이동은 아래 단계별로 진행됩니다.

1. 고객 확인

객실 이동 요청한 고객을 확인합니다.

2. 객실 요청 원인 파악

고객 요청 이유를 확인하고, 우선적으로 불만사항을 해결합니다.

• 불만사항 해결 후에도 이동이 필요한 경우 요청 진행

3. 객실 이동 진행

Reservation의 Room 클릭 후 이동할 객실 선택

• ★CLEAN(청소 완료) 객실로 지정

• 객실 이동 완료 처리

4. 하우스키핑 인계

객실 이동 정보를 하우스키핑 팀에 전달하여 청소 및 준비 진행.

5. Res Note 작성

객실 이동 완료 후, Reservation Note에 이동 내역 기록.
